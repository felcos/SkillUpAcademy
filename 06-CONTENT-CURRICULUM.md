# SkillUp Academy — Contenido y Currículo

## Estructura de Contenido por Área

Cada área tiene 3 niveles. Cada nivel tiene 4-6 lecciones.
El seed de la base de datos debe incluir **todo el contenido del Nivel 1** de las 6 áreas y la **estructura (título + tipo)** de los niveles 2 y 3.

## Notas para Claude Code

1. **El contenido textual completo** (párrafos de teoría, preguntas de quiz con opciones y feedback, descripciones de escenarios con opciones y consecuencias) debe generarse como parte del seed data. No dejar placeholders.

2. **El contenido debe ser realista y profesional:** usar ejemplos de oficina, reuniones, proyectos, clientes. Nada genérico o infantil.

3. **Los quizzes** deben tener exactamente 4 opciones, 1 correcta, y cada opción debe tener su propio feedback específico explicando por qué es correcta o incorrecta.

4. **Los escenarios** deben tener 3 opciones: una positiva (10 pts), una neutra (5 pts) y una negativa (0 pts). Cada opción tiene feedback detallado.

5. **Los audioScripts** deben estar escritos en tono conversacional, como si la IA Aria estuviera explicándote en persona. Más cortos que el contenido escrito.
